📋 Overview

Microsoft Graph Email Sender is a powerful C++ application that enables seamless email sending with attachments through Microsoft Graph API. The application implements OAuth 2.0 authentication and intelligently handles files of all sizes - from tiny text files to large attachments up to 150MB.

✨ Features

  • 🔐 OAuth 2.0 Authentication - Secure client credentials flow with Azure AD
  • 🔄 Automatic Token Management - Token caching and auto-refresh
  • 📎 Smart Attachment Handling - Files ≤3MB direct upload, >3MB chunked upload
  • 📦 Multiple File Support - Send unlimited attachments per email
  • 🔧 Visual Studio 2008 Compatible - Pure Win32 API, no dependencies
  • ⚡ High Performance - Native code with efficient memory management

📋 Prerequisites

Requirement Details
Visual Studio 2008 or later
Windows XP/Vista/7/8/10/11
Libraries WinHTTP.lib, Crypt32.lib
Azure Account Required for OAuth 2.0

🔧 Azure AD Setup

  1. Go to Azure Portal
  2. Register an application
  3. Add Mail.Send permission
  4. Create client secret
  5. Copy Client ID, Tenant ID, and Secret
