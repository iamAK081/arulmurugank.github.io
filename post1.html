📋 Overview

Microsoft Graph Email Sender is a powerful C++ application that enables seamless email sending with attachments through Microsoft Graph API. The application implements OAuth 2.0 authentication and intelligently handles files of all sizes - from tiny text files to large attachments up to 150MB.

✨ Features

Feature Description
🔐 OAuth 2.0 Secure authentication with Azure AD
🔄 Auto Token Refresh Never worry about expired tokens
📎 Smart Attachments Files ≤3MB direct, >3MB chunked upload
📦 Multiple Files Send unlimited attachments
🔧 VS 2008 Pure Win32 API, no dependencies

📋 Prerequisites

  • Visual Studio 2008 or later
  • Windows SDK
  • Azure AD account with Mail.Send permission

🔧 Quick Setup

  1. Register app in Azure AD
  2. Add Mail.Send permission
  3. Create client secret
  4. Update credentials in main()
  5. Build and run
