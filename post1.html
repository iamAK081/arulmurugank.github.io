📋 Introduction

Microsoft Graph Email Sender is a powerful C++ application developed by Arulmurugan K that enables seamless email sending with attachments through Microsoft Graph API. The application implements OAuth 2.0 authentication and intelligently handles files of all sizes - from tiny text files to large attachments up to 150MB.

🎯 Perfect For:

  • Enterprise automation systems
  • Automated reporting tools
  • Backup notification systems
  • Batch email processing

✨ Key Features

  • 🔐 OAuth 2.0 Authentication - Secure client credentials flow with Azure AD
  • 🔄 Automatic Token Management - Token caching and auto-refresh 5 minutes before expiry
  • 📎 Smart Attachment Handling - Files ≤3MB direct upload (Base64), >3MB chunked upload (5MB chunks)
  • 📦 Multiple File Support - Send unlimited attachments per email
  • 🔧 Visual Studio 2008 Compatible - Pure Win32 API, no MFC/ATL dependencies
  • ⚡ High Performance - Native code with efficient memory management
  • 📝 MIME Type Detection - Automatic content-type identification
  • 🛡️ Error Recovery - Comprehensive error handling with retry logic

📋 Prerequisites

Requirement Details
Visual Studio 2008 or later
Windows XP/Vista/7/8/10/11
Libraries WinHTTP.lib, Crypt32.lib
Azure Account Required for OAuth 2.0

🔑 Azure AD Setup

  1. Register Application - Go to Azure Portal → App registrations → New registration
  2. Add Permissions - Add Microsoft Graph → Application permissions → Mail.Send
  3. Create Secret - Certificates & secrets → New client secret
  4. Copy Values - Save Client ID, Tenant ID, and Client Secret

🚀 Usage Example

// Configure your credentials
std::wstring clientId = L"your-client-id";
std::wstring clientSecret = L"your-client-secret";
std::wstring tenantId = L"your-tenant-id";
std::wstring userId = L"your-email@company.com";

// Create attachments list
std::vector<std::wstring> attachments;
attachments.push_back(L"C:\\reports\\document.pdf");
attachments.push_back(L"C:\\backups\\data.zip");

// Send email
EmailSender sender(clientId, clientSecret, tenantId, userId);
sender.SendEmail(L"recipient@company.com",
L"Test Subject",
L"Email body",
attachments);
